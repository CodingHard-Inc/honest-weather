
## Getting started

**Clone the repo**

```
$ git clone git@github.com:CodingHard-Inc/honest-weather.git
$ cd honest-weather
```

**Install dependencies**

```
$ npm install
```

**Start Run**

```
$ npm start
```

## Build Package for Production

Build once for production. (minified)

```
$ npm run build
```

*ps: Builds into `public/` as `bundle.js`*
